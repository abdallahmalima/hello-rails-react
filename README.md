# 📖 Hello React Rails <a name="about-project"></a>

This project is a demo of how to use react and rails as a single project by using Webpack bundler.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

  <ul>
    <li><a href="https://rubyonrails.org/">Rails</a></li>
    <li><a href="https://www.postgresql.org/">PostgreSQL</a></li>
  </ul>

### Key Features <a name="key-features"></a>

- SPA functionality with React
- Handle state management with Redux
- Integrate Rails with React using Api controllers


## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- IDE or code editor: **VsCode, atom, sublime**...
- To have [Ruby](https://www.ruby-lang.org/en/) and [Git](https://git-scm.com/) installed on your system.
- To have [Node.js](https://nodejs.org/) installed on your system.

After installing Ruby, run the following command to install rails.-

```sh
 gem install rails
```

### Setup

1. Clone this project on your local machine using the following command in your terminal:

```sh
  git clone git@github.com:abdallahmalima/hello-rails-react.git
```

2. Inside the project directory, install the project's dependencies

   ```
   bundle install
   ```

3. Setup the app's database
   ```
   rails db:create
   rails db:migrate
   rails db:seed
   ```

### Install

Run the following command to install all project's dependencies:

```sh
  cd hello-rails-react
  bundle install
  node install
```

### Usage

To run the project, execute the following command:

Example command:

```sh
  rails server
```

OR

```sh
  rails s
```
